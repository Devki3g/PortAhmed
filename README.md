## نظرة عامة

قالب Portfolio احترافي ومتكامل مبني باستخدام أحدث تقنيات الويب. يتميز بتصميم عصري جذاب، أداء عالي، ودعم كامل للغة العربية. مناسب للمطورين، المصممين، المسوقين، وأي محترف يرغب في عرض أعماله بشكل احترافي.

![Preview](preview.gif)

##  المميزات الرئيسية

### التصميم والواجهة
-  تصميم عصري وجذاب بتدرجات لونية جميلة
-  تأثيرات زجاجية (Glass Morphism) عصرية
-  دعم كامل للغة العربية RTL
-  وضع ليلي/نهاري مع حفظ التفضيلات
-  خطوط عربية محسنة (Cairo)
-  أيقونات احترافية من React Icons

### الأداء والتقنيات
-  مبني على React 18 مع Vite لأداء فائق السرعة
-  Tailwind CSS v4 للتصميم السريع والمتجاوب
-  Framer Motion للحركات السلسة
-  Zustand لإدارة الحالة
-  تصميم متجاوب بالكامل مع جميع الأجهزة
-  تحميل كسول للصور والمكونات
-  تحسين محركات البحث SEO

### الأقسام والمكونات
-  **القسم الرئيسي**: مع تأثيرات حركية وعداد إحصائيات
-  **نبذة عني**: معلومات شخصية وخبرات
-  **المهارات**: عرض تفاعلي للمهارات مع أشرطة تقدم
-  **المشاريع**: معرض مشاريع مع فلاتر تصنيف
-  **الخبرات**: تسلسل زمني للخبرات العملية
-  **آراء العملاء**: شهادات وتقييمات
-  **نموذج التواصل**: مع التحقق من البيانات

### سهولة التخصيص
-  ملف بيانات مركزي واحد للتخصيص
-  هيكل ملفات منظم وسهل الفهم
-  مكونات قابلة لإعادة الاستخدام
-  توثيق شامل وخطوة بخطوة
-  كود نظيف مع تعليقات توضيحية
